网站搭建方案

@TBLGSn
2021-05-18
网站

网站的域名由之前的 www.hero-lu.top 变成了 www.tblgsn.cn, 网站的主题也由之前的 cards 变为了 bufferfly.
网站采用以下的方案进行搭建:

  • 阿里云OSS 储存静态文件
  • 阿里云 CDN 解析 进行加速
  • github 作为源仓库
  • 利用 github action 进行监测，一旦push 时，action 就会自动执行，将生成的文件部署到另一个叫做 tblgsn.github.io 的仓库，同时推送到阿里云 OSS
  • 阿里云免费的SSL证书
  • 在 OSS 中设置’强制将 http 转为 https’
  • 使用 hexo-baidu-url-submit 进行自动提交
  • 已备案，因为是 oss，所以不能申请备案服务号，所以买了一个，没想到也行
